# FishChat

Hook WeChat.app on non-jailbroken devices.

## Features

- [x] 屏蔽群消息，好友消息（对方无察觉）
- [x] 关闭『发现』页面的『朋友圈』、『购物』和『游戏』入口
- [x] 修改微信运动步数
- [x] 去除各种小红点提示
- [x] 设置夜间模式
- [x] 阻止撤回消息
